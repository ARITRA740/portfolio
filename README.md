# Aritra Mondal Portfolio

Modern 3D portfolio built with Next.js App Router, TypeScript, Tailwind CSS, React Three Fiber, Drei, Framer Motion, and Lucide React.

## Setup

Place these images in `public`:

- `image_7bd387.jpg`
- `image_7bcfc4.jpg`

Then run:

```bash
npm install && npm run dev
```

Open `http://localhost:3000`.

## Deploy To Vercel

1. Initialize git, commit the project, and push it to a new GitHub repository:
   `git init -b main && git add . && git commit -m "Initial portfolio" && git remote add origin YOUR_GITHUB_REPO_URL && git push -u origin main`
2. Go to Vercel, choose `Add New Project`, import the GitHub repository, and keep the default Next.js settings.
3. Click `Deploy`; Vercel will build with `npm run build` and publish the free production URL.
